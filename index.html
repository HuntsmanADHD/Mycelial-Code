What is Mycelial?

Forget sequential programming. You're thinking in networks now. üåø

Instead of...

  • Sequential code (step 1, 2, 3...)
  • Centralized control
  • Complex coordination
  • Brittle dependencies

You write...

  • Agents that listen for signals
  • Distributed decision-making
  • Emergent behavior

hello_world.mycelial 
network HelloWorld {
  frequencies {
    greeting { name: string }
    response { message: string }
  }

  hyphae {
    hyphal greeter {
      on signal(greeting, g) {
        emit response {
          message: format("Hello, {}!", g.name)
        }
      }
    }
  }

  topology {
    fruiting_body input
    fruiting_body output
    spawn greeter as G1
    socket input -> G1 (frequency: greeting)
    socket G1 -> output (frequency: response)
  }
}
