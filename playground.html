Mycelial Playground Help

Keyboard Shortcuts

  • Ctrl/Cmd + Enter - Compile
  • Ctrl/Cmd + S - Save (local storage)
  • Ctrl/Cmd + F - Find in editor
  • Ctrl/Cmd + Z - Undo
  • Ctrl/Cmd + Y - Redo

Output Tabs

  • Tokens: Lexical analysis - see how code is tokenized
  • AST: Abstract Syntax Tree - parse tree structure
  • Assembly: Generated x86-64 assembly code
  • Binary: Binary file information and sections
  • Console: Compilation messages and errors
